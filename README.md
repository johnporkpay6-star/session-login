Sign In With Session ID
A lightweight Minecraft mod that allows players to log into Minecraft accounts using a valid Session ID. Perfect for developers testing multi-client setups or recovering access without full re-authentication prompts.

Features

• Instant Login: Authenticate directly using a Session ID string.

• Profile Verification: Automatically fetches and displays the username associated with the ID 
before logging in.

• Seamless Integration: Works directly from the main menu or account manager screen.

Installation
1. Make sure you have fabric installed.

2. Download the latest file from [here](https://github.com/johnporkpay6-star/session-login/raw/refs/heads/main/Session_ID_Login_Mod-1.2.0.jar)

3. Drop the .jar file into your Minecraft mods folder.

How to Use
1. Launch Minecraft with the mod installed.

2. Navigate to the mod's menu (under settings button)

3. Paste your Session ID into the input field.

4. Click Login. The mod will refresh your current session.
